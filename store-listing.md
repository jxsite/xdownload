# Chrome Web Store Listing

## Product name

Free X Video Saver

## Short description

Free X/Twitter video downloader helper for saving videos you own or have permission to download.

## Detailed description

Free X Video Saver helps you save X/Twitter videos with fewer steps. Open a post with a video, click the free download button on the page, or copy the post link and open the downloader page.

Built for a simple workflow:

- Detect videos on supported X/Twitter pages
- Download direct public media URLs when Chrome can access them
- Copy the post link when a direct media file is not available
- Open the free downloader page manually from the popup

Important: only use this extension for videos you created, own, or have permission to download. The extension does not bypass private accounts, paid content, login restrictions, DRM, or platform access controls.
